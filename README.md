# Identifying-world-writable-and-SUID-files
## Linux Permissions Audit  
A lightweight audit script for analyzing file and directory permissions across Linux systems. Helps identify weak ACLs, misconfigurations, and potential privilege‑escalation paths.
#### A practical script for auditing user/group permissions, SUID binaries, shadow file exposure, and privilege‑related anomalies — ideal for security labs and hardening workflows.
